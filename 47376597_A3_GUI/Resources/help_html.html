Help Section

    How to use this application

    ADD NEW CONTAINERS:

    1. Decide if you want to create either a BOX container or a CYLINDER container.
    2. Enter all the information requiered for the specific container you want to add.
    3. Click on the ADD BOX or ADD CYLINDER button to add the new container.
    4. You will only be allowed to add a container if all the information is entered.
    5. When you add a container, it will be listed in the UNALOCATED containers area.
    6. This is where all the unalocated containers are displayed after they were created.

    MOVE CONTAINERS TO PALLET:

    1. Once you created a container, you will be able to select the container you want to move to a pallet.
    2. Select the number for the pallet you want to move the container to.
    3. Once you selected the number, you can click the add MOVE TO PALLET button.
    4. The selected container will be removed from the list and be allocated to the specified pallet.

    MAKE A BACKUP:

    1. Once you have a list of unallocated containers, you'll be able to make a backup of the list when needed.
    2. Click on the BACKUP button on the screen to make a backup of the list.
    3. You can also use the backup button on the toolbar or the backup button in the menu.

    RESTORE A BACKUP:

    1. When you want to restore the list of the backup you made, you click on the restore button in the toolbar.
    2. The list will be displayed in the unallocated container area.
    3. You can also use the restore button on the menu.

    SEND THE PALLET LIST TO THE SERVER:

    1. Once you need to send the pallet list to the server, you click on the POST XML tab.
    2. In the POST XML tab, you will see the list of pallets have been displayed in XML form, ready to be sent to the server.
    3. Click on the POST XML TO NETWORK button to send the list of pallets to the server.
    4. REMEMBER: the network program should already be running and listening on the port number.
    5. You can also click the POST TO XML button in the toolbar or the button in the menu.
